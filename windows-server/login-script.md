# Login script

Pour mettre en place un login script, il y a trois étapes à réaliser :

1. Naviguer, sur le serveur, jusqu’au répertoire :

   ```text
   C:\Windows\Sysvol\Sysvol\<nom_du_domaine>\Script
   ```

   C’est ici que doit être créé le script.

2. Créer le script proprement dit.

3. Indiquer, dans le profil de tous les utilisateurs, le nom du script qui devra être exécuté au login.
